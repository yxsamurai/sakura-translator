# 🌸 Sakura Translator（桜トランスレーター）

> シンプルでミニマルな Chrome 翻訳拡張機能。Webページ上のテキストにホバーまたは選択するだけで、単語と文章を即座に翻訳します。

**[English](README_EN.md)** | **[中文](README_CN.md)**

---

## ✨ 機能

- **ホバー / 選択翻訳** — 2つの選択モードから選べます：
  - **ホバーモード**：テキストにマウスを合わせ、修飾キーを押すだけで自動選択＆翻訳
  - **手動モード**：テキストをドラッグで選択し、修飾キーを押して翻訳をトリガー
- **スマート検出** — 選択したのが単語か文章かを自動判別
- **辞書モード** — 単語翻訳では発音記号、品詞、定義、例文など詳細な情報を表示
- **文章モード** — 文章翻訳ではシンプルで直接的な訳文を表示
- **30以上の言語** — 英語、中国語（簡体字/繁体字）、日本語、韓国語、フランス語、ドイツ語、スペイン語、ポルトガル語、ロシア語、アラビア語、ヒンディー語、イタリア語、オランダ語、タイ語、ベトナム語、インドネシア語、マレー語、トルコ語、ポーランド語、ウクライナ語、スウェーデン語、デンマーク語、フィンランド語、ノルウェー語、ギリシャ語、チェコ語、ルーマニア語など
- **ダークモード** — システムテーマに自動対応
- **カスタムホットキー** — 修飾キー（Ctrl / Alt / Shift）を単語・文章選択用にカスタマイズ可能（自動競合防止機能付き）
- **ポップアップ翻訳** — 拡張機能アイコンをクリックして手動翻訳入力ボックスを開く
- **設定不要** — 無料APIを使用し、そのまますぐに使えます（APIキー不要）

## 📦 インストール

1. リポジトリをクローンまたはダウンロード：
   ```bash
   git clone https://github.com/user/sakura-translator.git
   ```
2. アイコンを生成（初回のみ）：
   ```bash
   node generate-icons.js
   ```
3. Chrome を開き、`chrome://extensions/` にアクセス
4. 右上の**デベロッパーモード**を有効にする
5. **パッケージ化されていない拡張機能を読み込む**をクリックし、このプロジェクトフォルダを選択
6. インストール完了！ツールバーの **🌸** アイコンを探してください

## 🎯 使い方

### ページ内翻訳（ホバーモード — デフォルト）

1. Webページ上の任意のテキストにマウスを**ホバー**
2. **Ctrl** キーを押すと、カーソル下の単語を選択＆翻訳
3. **Alt** キーを押すと、文章全体を選択＆翻訳
4. フローティングポップアップに翻訳結果が表示されます
5. **Escape** キーまたは他の場所をクリックでポップアップを閉じる

### ページ内翻訳（手動モード）

1. Webページ上の任意のテキストを**ドラッグ**して選択
2. 修飾キー（デフォルト：**Ctrl**）を押しながら選択を離す
3. または先にテキストを選択してから修飾キーを押す
4. フローティングポップアップに翻訳結果が表示されます

### ポップアップ翻訳

1. ツールバーの拡張機能アイコンをクリック
2. 入力ボックスにテキストを入力または貼り付け
3. **Ctrl+Enter** を押すか、**Translate** ボタンをクリック

### 設定

ポップアップの ⚙️ 歯車アイコンをクリックして設定：
- **ソース言語 / ターゲット言語** — 30以上の言語から選択、ワンクリックで入れ替え可能
- **選択モード** — ホバーモードと手動モードを切り替え
- **ホットキー** — 単語/文章選択の修飾キーをカスタマイズ（キーの競合は自動防止）

すべての設定は即座に自動保存されます。

## 🔌 使用API

| API | 用途 | 料金 |
|-----|------|------|
| [Google Translate](https://translate.googleapis.com/) | 多言語翻訳、拡張辞書データ | 無料 |
| [Free Dictionary API](https://dictionaryapi.dev/) | 英単語の定義、発音記号、例文 | 無料、無制限 |

## 📁 プロジェクト構成

```
sakura-translator/
├── manifest.json           # Chrome 拡張機能 Manifest V3
├── background.js           # Service Worker（API呼び出し、言語ルーティング）
├── content/
│   ├── content.js          # コンテンツスクリプト（選択検出、ポップアップUI）
│   └── content.css         # フローティングポップアップのスタイル
├── popup/
│   ├── popup.html          # 拡張機能ポップアップページ
│   ├── popup.js            # ポップアップロジック＆設定管理
│   └── popup.css           # ポップアップスタイル
├── icons/
│   ├── icon16.png
│   ├── icon48.png
│   └── icon128.png
├── utils/
│   ├── detector.js         # 単語/文章の検出
│   ├── translator.js       # 翻訳API抽象化レイヤー
│   └── md5.js              # MD5ユーティリティ
├── tests/
│   ├── background.unit.spec.js
│   ├── detector.unit.spec.js
│   ├── md5.unit.spec.js
│   ├── content-script.spec.js
│   ├── popup-ui.spec.js
│   └── fixtures.js
├── playwright.config.js    # テスト設定
├── generate-icons.js       # アイコン生成スクリプト
└── package.json
```

## 🧪 テスト

本プロジェクトは **Playwright** を使用してユニットテストとE2Eテストを実施しています。

```bash
# すべてのテストを実行
npm test

# ユニットテストのみ
npm run test:unit

# 拡張機能E2Eテスト
npm run test:extension

# ポップアップUIテストのみ
npm run test:popup

# コンテンツスクリプトテストのみ
npm run test:content
```
